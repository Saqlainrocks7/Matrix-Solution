# Matrix-Solution

## Logic Behind the Solution

### Step 1

The JSON represents: team → opponent → { W, L }

### Step 2 

A matrix means:

rows = team

columns = team

### Step 3

For each cell:

if row_team == col_team → "--"

else → "W-L" from row_team’s perspective
